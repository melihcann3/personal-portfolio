Hakkımda

Merhaba! Ben Melih CAN, İstanbul Topkapı Üniversitesi Yazılım Mühendisliği bölümü öğrencisiyim. Çok farklı yönlerde ve sektörde iş deneyimim oldu. Defalarca ekip olarak çalıştım ve yönettim. Daha sonrasında Yazılım merakım beni bu yönde ilerletti Teknolojiye ve problem çözmeye olan tutkumla bu alana yöneldim. Şu anda özellikle web geliştirme teknolojileri olan HTML, CSS ve yakında JavaScript üzerine yoğunlaşıyorum.

Bu portfolyo sitesi, Web Uygulamaları Geliştirme dersi vize projem kapsamında hazırlanmıştır. Amacım, öğrendiklerimi pratik etmek ve temiz, kullanışlı web siteleri oluşturma becerimi geliştirmektir.
